# DATA DOWNLOAD INSTRUCTIONS

Due to GitHub's file size limitations, the raw and processed data required to run these scripts (.csv files) are hosted securely on Zenodo.

**WARNING: MASSIVE UNCOMPRESSED DATASET (>70 GB)**

While the downloaded `.zip` archive is highly compressed, extracting its contents will generate over 70 GB of `.csv` files. Please ensure you have sufficient free disk space on your machine before attempting to unzip the folder.

## How to get the data:
1. Download the highly compressed data archive from the following Zenodo repository:
   -> [PEGA_AQUÍ_TU_ENLACE_DOI_DE_ZENODO]
   
2. Extract the contents of the downloaded archive (Make sure you have >70 GB of free space). 

3. Place all the extracted `.csv` files directly into this `DATA/` folder. 

Once the files are placed here, you can run all the R scripts in the repository without needing to change any file paths.
